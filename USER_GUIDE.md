# Zento AI - User Guide

## Getting Started

### 1. Registration
1. Visit the application and click on the "Sign Up" tab
2. Choose a username (letters, numbers, and underscores only)
3. Create a secure password
4. Confirm your password
5. Click "Create Account"

**Note:** The first user to register automatically becomes an administrator.

### 2. Login Options

#### Username & Password
- Enter your username and password
- Click "Login"

#### Google SSO
- Click "Google SSO" button
- Follow the authentication flow

#### Phone Login
- Enter your phone number (with country code, e.g., +1234567890)
- Click "Send OTP"
- Enter the OTP code you receive
- Click "Verify OTP"

## Using the Chat Interface

### Starting a New Chat
1. Click the "New Chat" button in the sidebar
2. Type your message in the input box at the bottom
3. Press Enter or click the Send button

### Sending Messages
- **Text Only:** Type your message and press Enter
- **With Image:** Click the image icon, select an image, then type your message
- **Multi-line:** Use Shift+Enter for new lines

### Supported Languages
Zento AI automatically detects and responds in multiple languages including:
- English
- Uzbek
- Russian
- And many more!

## Troubleshooting

### Message Not Sending
- Check your internet connection
- Ensure you're logged in
- Try refreshing the page

### Image Upload Failed
- Check file size (must be under 1MB or will be compressed)
- Verify file format (JPEG, PNG, GIF, WEBP, AVIF)
- Ensure filename contains only English letters and numbers

### Streaming Stopped
- Wait for the response to complete
- If stuck, try regenerating the response
- Refresh the page if the issue persists

### Theme Not Saving
- Clear your browser cache
- Check browser localStorage settings
- Try a different browser

## Privacy & Security

### Your Data
- All chats are private and user-specific
- Only you and administrators can see your chat history
- Messages are encrypted in transit
- Images are stored securely in isolated folders

### Account Security
- Use a strong, unique password
- Don't share your login credentials
- Log out when using shared devices
- Contact an administrator if you suspect unauthorized access

## Best Practices

1. **Start Fresh:** Create a new chat for different topics
2. **Organize:** Rename chats with descriptive titles
3. **Clean Up:** Delete old chats you no longer need
4. **Be Clear:** Write clear, specific questions
5. **Iterate:** If the response isn't what you need, ask follow-up questions

## Getting Help

### Common Questions

**Q: How do I become an admin?**
A: The first user to register is automatically an admin. Other users can be promoted by existing admins through the Admin Panel.

**Q: Can I edit my messages?**
A: Currently, messages cannot be edited after sending. You can send a new message to clarify or correct.

**Q: How long is my chat history saved?**
A: Chat history is saved indefinitely unless you delete it.

**Q: Can I export my chats?**
A: Export functionality is planned for a future update.

**Q: Is there a message limit?**
A: There's no hard limit on message length, but very long messages may take longer to process.

### Need More Help?
Contact your system administrator for additional support or feature requests.
